## 📌 Setup & Installation

### Prerequisites
Ensure you have the following installed:
- **Node.js & npm**
- **TypeScript**
- **Solana CLI**
- **gRPC Libraries**

### Installation Steps
```sh
- git clone https://github.com/your-username/solana-grpc-sniper-bot.git
- cd solana-grpc-sniper-bot
- npm install
```

### Running the Bot
```sh
npm run start
```
